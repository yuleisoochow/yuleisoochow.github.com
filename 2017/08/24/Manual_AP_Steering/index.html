Manual AP Steering

Posted on 2017-08-24 | In Wireless
Words count in article 517 | Reading time 3

在debug模式可以用下面的命令手动将本AP上的WIIF client切换到另一个指定的AP上
steerexec steer
Example:
用td s命令show出的信息如下

此时，客户端2C:33:7A:85:8D:33连在该rootAP的2.4G上，
命令steerexec steer 2C:33:7A:85:8D:33 0 3 0 可以将该客户端切换到第一个RE的2.4G上
命令steerexec steer 2C:33:7A:85:8D:33 1 3 0 可以将该客户端切换到第二个RE的2.4G上
